## [NPM Scripts](#npm-scripts)
Run only tests.
> npm run test

Run only project building.
> npm run build

Run only dev server.
> npm run dev

Run only project preview (you need have a compiled project).
> npm run preview

Run tests and then project building.
> npm run build:test

Run project building and then project preview.
> npm run build:preview

Run tests then project building and then project preview.
> npm run build:test:preview
